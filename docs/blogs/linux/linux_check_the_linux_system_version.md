## 一、查看 Linux 内核版本命令（两种方法）：

```bash
cat /proc/version
```

```bash
uname -a
```

## 二、查看 Linux 系统版本的命令（3 种方法）

### 适用于所有的 Linux 发行版，包括 RedHat、SUSE、Debian…等发行版

```bash
lsb_release -a
```

### 只适合 Redhat 系的 Linux

```bash
lsb_release -a
```

### 适用于所有的 Linux 发行版

```bash
lsb_release -a
```
