1、mvn compile 编译,将 Java 源程序编译成 class 字节码文件。
2、mvn test 测试，并生成测试报告
3、mvn clean 将以前编译得到的旧的 class 字节码文件删除
4、mvn pakage 打包,动态 web 工程打 war 包，Java 工程打 jar 包。
5、mvn install 将项目生成 jar 包放在仓库中，以便别的模块调用
